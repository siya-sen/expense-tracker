# Expense Tracker - Django Project

A simple **Expense Tracker** web application built with **Django** that allows users to manage their personal expenses efficiently.

## Features

- **Add Expenses**: Add new expenses with amount, category, date, and description.  
- **Edit Expenses**: Update any existing expense.  
- **Delete Expenses**: Remove unwanted expenses.  
- **Dashboard**: View all expenses with total calculation.  
- **Category Management**: Select predefined categories for expenses.  
- **User Authentication**: Login/logout support for secure access (optional if added).  


## Tech Stack

- Python 3.12  
- Django 5.0  
- HTML/CSS (for templates)  
- SQLite (default database)
